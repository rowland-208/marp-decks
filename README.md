Marp-Decks is a collection of claude skills to build better slide decks with marp.

Check out these example decks, built with marp-decks and hosted on github pages:

[https://rowland-208.github.io/marp-decks/](https://rowland-208.github.io/marp-decks/)

## Getting Started

1. `$ git clone https://github.com/rowland-208/marp-decks.git && cd marp-decks`
2. [Install Claude Code](https://code.claude.com/docs/en/quickstart)
3. `$ claude 'create a slide deck about whales'`
4. (Optional) Fork this repo and enable github pages to view your published decks online

## Features

- **Live rendering** - See your changes instantly as you edit, no manual rebuilds needed
- **Image generation and publication skills** - Generate custom images and publish decks with built-in slash commands
- **Slides in version control** - Every deck is plain Markdown, so you get full git history, diffs, and collaboration for free
- **Claude Code native** - Designed from the ground up to work with Claude Code as your slide-building copilot
- **Skills to build new skills** - Extend the toolkit by creating your own custom skills
- **Publish to GitHub Pages** - Go from local slides to a live URL with a single command

## Suggested Workflow

1. Ask Claude to create a new deck
2. Open the `file://` link Claude provides to view changes live
3. Alt-tab between your terminal and browser as you iterate on slides
4. Use the `/publish` slash command to publish your deck to github pages
