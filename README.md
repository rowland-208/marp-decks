# marp-decks

Presentation slide decks built with [Marp](https://marp.app/), a Markdown-based slide framework.

## Getting Started

This project is designed to be used with [Claude Code](https://claude.ai/code).

1. Clone the repo and open it in your terminal
2. Install Claude Code if you haven't already
3. Run `claude` and start building
4. (Optional) Fork this repo and enable GitHub Pages to view your published decks online

## Suggested Workflow

1. Start `claude` in the repo root
2. Ask Claude to create a new deck (e.g. "create a deck about laser cutters")
3. Open the `file://` link Claude provides in your browser and go full screen
4. Make your terminal window always-on-top so you can iterate while viewing the slides
5. Use the `/publish` slash command to publish your deck to GitHub Pages
